Version 2.5

  • Multi-app launching — assign several apps to a single gesture or tap pattern; they all open at once
  • Open URLs — map any trigger to one or more web links; they open in your default browser
  • Combined actions — mix apps and URLs freely in the same trigger (e.g. open Figma + your project dashboard in one tap)
  • Improved action editor — live preview of what will open, circle checkboxes for apps, inline URL input
  • Fixed "Input Monitoring required" button — now opens System Settings directly when permission was previously denied
